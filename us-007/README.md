# US-007 : Navigation dans l’arborescence Linux

## Objectif

Apprendre à naviguer dans le système Linux avec les commandes de base.

---

## Emplacement actuel

```bash
pwd

##  Revenir au dossier parent
cd ..

## Chemin absolu
commence par /

##  Dossiers importants
cd /etc

##  Auto-complétion TAB
cd /ho[TAB]
```
